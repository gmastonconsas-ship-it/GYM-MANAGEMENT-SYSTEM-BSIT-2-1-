## 🛠 How to Setup & Run
To run this system on your local machine, follow these steps:

1. **Database Import**
   * Open `localhost/phpmyadmin`.
   * Create a new database (e.g., `gym_system`).
   * Import the `.sql` file included in this repository.

2. **Project Placement**
   * Move the entire project folder into your XAMPP `htdocs` directory.

3. **Configuration**
   * Check the `config.php` (or your connection file).
   * Ensure the database name matches what you created in Step 1.
   * Default XAMPP settings: `User: root`, `Password: (empty)`.

4. **Launch**
   * Open your browser and go to: `localhost/your_folder_name/login.php`

## ✨ Key Features
* **Member Management:** Register, update, and track gym members.
* **QR Scanner UI:** A simulated GCash-style scanning interface for modern UX.
* **Membership Status:** Real-time tracking of active vs. expired subscriptions.
* **Database Driven:** Built using PHP and MySQL for reliable data storage.
