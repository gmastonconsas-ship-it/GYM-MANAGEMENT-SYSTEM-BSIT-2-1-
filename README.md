## 🛠 How to Setup & Run
1. **Database:** Import the `.sql` file into your PHPMyAdmin.
2. **Placement:** Move this folder to your XAMPP `htdocs` directory.
3. **Access:** Open your browser and go to `localhost/your_folder_name/login.php`.

## ✨ Key Features
* Member Registration & Management
* QR Code Scanning UI (Simulation)
* Membership Status Tracking
